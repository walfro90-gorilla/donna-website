# Product Overview

Do√±a Repartos is a food delivery platform connecting local restaurants, customers, and delivery drivers in Spanish-speaking markets.

## Core Features

- Multi-role registration system (clients, restaurants, delivery agents)
- Role-based dashboards with authentication
- Real-time form validation with Supabase backend
- Google Maps integration for address autocomplete
- Cookie consent management (GDPR compliance)

## User Roles

- **Clients**: Order food from local restaurants
- **Restaurants (Socios)**: Manage menu and receive orders
- **Delivery Agents (Repartidores)**: Accept and complete deliveries
- **Admin**: Platform management and oversight

## Language

All user-facing content is in Spanish. Use Spanish for:
- UI labels and messages
- Error messages and validation feedback
- Comments in user-facing components
- ARIA labels and accessibility text

Internal code comments and technical documentation can be in English.
